Welcome to the Looty homepage.

Looty is a tool for Path of Exile that makes inventory searching easier.

It is an extension for Google Chrome that downloads inventory data from the Path of Exile website. You need to be logged into the pathofexile.com site in order for it to be able to download the data.

Source code available on github.

Click to download the chrome extension. Download chrome extension.crx

After downloading, to install it, open the extensions view in Chrome: under menu -- tools -- extension Then simply drag and drop the .crx file you downloaded into the extensions view. Chrome will ask for permission to install it. If you are upgrading just delete the old version first.
